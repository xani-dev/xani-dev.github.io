Real Time Bus Tracker

Project Date March 9th, 2021
Technologies MapBox, Javascript
Project for MIT xPRO
/Repo link GitHub

This project provides a visual real-time tracking display of the Route #1 of Boston, running between Harvard and Nubian.

See it in action here. Remember you'll need to add your MapBox Key to be able to display your map.

Displays a colored set of markers showing all the buses running this route in real time. Info can also be updated by pressing the button located on the top right corner, if needed.
